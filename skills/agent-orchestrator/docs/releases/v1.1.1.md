# v1.1.1 Release Notes

## Runtime Default Matrix

| Variable | Default |
|----------|---------|
| `OPENCLAW_AGENT_TIMEOUT_SECONDS` | `600` |
| `ORCH_WORKER_JOB_TIMEOUT_SECONDS` | `2400` |
| `ORCH_RUNNING_STALE_SECONDS` | `300` |
| `ORCH_HEARTBEAT_LOG_SECONDS` | `30` |
| `ORCH_WORKER_MAX_CONCURRENCY` | `2` |
| `ORCH_MAX_PARALLEL_TASKS` | `2` |

## Notes

- `ORCH_AGENT_MAX_CONCURRENCY` is supported as a legacy alias for worker concurrency.
- Run `python3 scripts/check_runtime_defaults.py` to detect runtime/docs drift.
